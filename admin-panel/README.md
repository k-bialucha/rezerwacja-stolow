# Admin Panel (aplikacja React)
## Wymagania:
- Node.JS v6.x
- npm v4.x (dostarczane razem z Node.JS)
## Uruchomienie
Pobranie pakietów:

    npm install
Uruchomienie aplikacji w œrodowisku Dev:

    npm start
Aplikacja uruchomi siê automatycznie pod adresem http://localhost:3000/.

Dla Windows dostêpny jest skrypt **run.ps1**, który wykonuje oba polecenia.
